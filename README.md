## 🚀 Quick Start

### Prerequisites
- **macOS** (tested on macOS with Apple Silicon)
- **Python 3.11+**
- **Homebrew** for Tor installation
- **Chrome/Chromium** browser

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/jedixcom/xcom-advanced-web-site-hitter.git
cd xcom-advanced-web-site-hitter
```

2. **Install Tor via Homebrew**
```bash
brew install tor
```

3. **Create Python virtual environment**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

4. **Install Python dependencies**
```bash
pip install -r requirements.txt
```

5. **Run the application**
```bash
python tor_menu.py
```

## 🔧 Troubleshooting

### Common Issues

**Tor Connection Failed**
```bash
# Restart Tor service
brew services restart tor
# Check Tor status
brew services list | grep tor
```

**Browser Crashes**
- Ensure Chrome/Chromium is installed
- Try headless mode for better stability
- Check available memory

**Permission Errors**
```bash
# Fix permissions
chmod +x tor_menu.py
# Reinstall dependencies
pip install --force-reinstall -r requirements.txt
```
